## 关键特性位置

### 身份验证
- `app.py` - `login()`, `auth_callback()` 路由
- `config.py` - MSAL 配置
- `templates/login.html` - 登录界面

### 会话管理
- `app.py` - `before_request()`, `login_required()` 装饰器
- `static/js/session.js` - 客户端计时器
- `config.py` - SESSION_TIMEOUT 配置

### 用户界面
- `templates/base.html` - AWS 风格布局
- `static/css/style.css` - 侧边栏和内容样式
- `templates/dashboard.html` - 四个功能卡片

### 功能模块
- `templates/page_a.html` - Alpha 配置 (资源管理)
- `templates/page_b.html` - Beta 部署 (部署管道)
- `templates/page_c.html` - Gamma 监控 (性能指标)
- `templates/page_d.html` - Delta 分析 (日志查询)

## 开发建议

### 添加新功能
1. 在 `app.py` 创建新路由
2. 在 `templates/` 创建新 HTML 文件
3. 在 `base.html` 中添加导航链接
4. 编辑 `static/css/style.css` 添加样式

### 修改样式
- 全局样式: 编辑 `static/css/style.css`
- 登录页面: 编辑 `static/css/login.css`
- 特定页面: 在相应 HTML 文件中的 `<style>` 块

### 扩展功能
- 添加数据库: 集成 SQLAlchemy
- API 开发: 在 `app.py` 添加 `@app.route()` 装饰的函数
- 前端交互: 编辑 `static/js/session.js` 或创建新 JS 文件
